# Government Scheme Eligibility Assistant

Finding applicable government schemes can be confusing and time-consuming. This project solves that problem by acting as an intelligent assistant that evaluates user information, analyzes eligibility conditions, and recommends the most relevant schemes instantly.

Instead of manually searching multiple websites, users simply answer a few questions and receive tailored results with explanations and application links.
